# CLAUDE.md — Somos Produtora

## Projeto
Sistema Integrado de Gestão Inteligente para a Somos Produtora — plataforma SaaS com CRM, ERP, gestão de projetos, agentes de IA e app mobile.

## Stack
- **Backend:** Python 3.12 + FastAPI
- **Frontend:** Next.js 15 + TypeScript + Tailwind CSS + ShadcnUI
- **Mobile:** React Native + Expo
- **Database:** PostgreSQL 16 + Redis 7 + Qdrant + MinIO
- **AI Agents:** CrewAI + LangChain (Claude 3.5 Sonnet, GPT-4o, Gemini Pro)
- **Infra:** Docker + Nginx + GitHub Actions
